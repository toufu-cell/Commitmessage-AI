# Commitmessage-AI

このプロジェクトは、コミットメッセージを自動生成するAIツールです。

## セットアップ

1. 仮想環境を作成：
   ```bash
   python -m venv venv
   ```

2. 依存関係をインストール：
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

1. 仮想環境を起動：
   ```bash
   gitchange
   ```

2. メインスクリプトを実行：
   ```bash
   python main.py
   ```

## 便利なエイリアス

- `gitchange`: 仮想環境を起動し、次のステップを表示

## 依存関係

- Python 3.x
- 必要なパッケージはrequirements.txtに記載
